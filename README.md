# TFM-Simulations
Repositorio con todas las simulaciones datos y demás del TFM. 

## Instalación:

nano install_all.sh
./install_all.sh
source ~/.bashrc


## Comprobación

root -l        # debería abrir ROOT
geant4-config --version   # debería devolver 11.2.2
cd ~/garfield-install/lib && ls libGarfield.so
